# Online Retail Sales EDA 📊
Exploratory Data Analysis on UK Online Retail dataset to identify sales trends, top products, and customer behavior using Python.

### 🎯 Business Questions Solved
1. Which countries generate the most revenue?
2. What are the top 10 best-selling products?
3. Which months have peak sales?
4. What % of orders are cancelled?

### 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
- **Tool:** Jupyter Notebook

### 📈 Key Insights Found
1. **UK contributes 82%+ of total revenue** – Major market focus area.
2. **Q4 (Oct-Dec) shows 40% higher sales** – Holiday season impact.
3. **'WHITE HANGING HEART T-LIGHT HOLDER'** is the top-selling product.
4. **24% transactions were cancellations** – Potential revenue loss area.
